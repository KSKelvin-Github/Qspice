# FRA Frequency Response Analyzer (Dual Transfer Functions)
* This is a variant with 2 transfer functions from the FRA Frequency Response Analyzer project
* More information can be found in the original project folder, including FRA accessories

## Files
* FRAnalyzer2KSK.qsym - symbol of FRA
* franalyzer2ksk.cpp - C++ code (only requires if you want to study or modify the code)

## Folders
* examples - all examples (copy .dll into working directory to execute example in Qspice)
